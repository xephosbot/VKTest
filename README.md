# VKTest

Тестовое приложение, отображающее список файлов на устройстве. Реализованные функции:
- [x] Отображение имени, размера, даты создания, иконки файла
- [x] Сортировка списка файлов
- [x] Сохранение хеш кодов, и отображение файлов измененных с момента последнего запуска
- [x] Открытие файлов
- [ ] Возможность поделиться файлом
